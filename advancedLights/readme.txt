+-----------------------------------------------------------------------------|
|  About                                                                      |
+-----------------------------------------------------------------------------|
  
  Theoretically supports 1.8.8-1.12.2
  Tested on 1.12.1
  
  This addon adds 9 new light sources, though users can add more provided the
  correct syntax is used.
  
  The Electric Lantern can be toggled on and off by right clicking it.
  
  The four coloured candles (white, brown, red, and ivory) can be doused by
  right clicking them while they're emitting light. To relight them, right
  click them with a Flint and Steel or a Fire Charge.
  
  
+-----------------------------------------------------------------------------|
|  How to Install                                                             |
+-----------------------------------------------------------------------------|
  
  Required dependencies:
  - Denizen (plugin)
  - GrandPartyUtilities Version Checker (script)
  Optional dependencies:
  - none
  
  When installing Denizen, please use the latest DEVELOPMENT version!
  
  If your server is still running, simply use this command to load the addon:
    /denizen reload scripts
  
  After this, you're done! Begin by running "/advancedlights help".
  
  
+-----------------------------------------------------------------------------|
|  Commands and Permissions                                                   |
+-----------------------------------------------------------------------------|
  
  COMMANDS:
    /advancedlights help
      - prints the help page
    
    /advancedlights give <item_name> (quantity) (player)
      - gives an AdvancedLight item, with the optional "quantity" and "player"
        arguments
  
  PERMISSIONS:
    gpa.advancedlights
    - allows use of /advancedlights
